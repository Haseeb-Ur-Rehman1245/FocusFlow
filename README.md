🚀 FocusFlow – Productivity Timer & Task Visualizer

✨ Stay focused. Build momentum. Ship faster.

FocusFlow is a modern Pomodoro productivity web app designed to help you manage deep work sessions, track tasks, and visualize your productivity — all wrapped in a clean, SaaS-level UI inspired by Notion & Linear.

🌟 Features

🕒 Pomodoro Timer

25-minute focus sessions

5-minute break cycles

Auto-switch between focus & break modes

🎯 Task Management

Add and manage tasks easily

Mark tasks as completed

Clean, distraction-free task UI

📊 Productivity Tracking

Session counter

Visual progress indicator

Real-time timer feedback

🎨 Modern UI/UX

Glassmorphism design

Animated circular progress ring

Smooth micro-interactions

🌌 Interactive Background

Animated moving stars using Canvas

Creates a calming deep-work environment

🌙 Dark / Light Mode

Toggle between themes instantly

Optimized for long productivity sessions

⚡ Performance Focused

Built with pure HTML, CSS, JavaScript

No frameworks, lightweight & fast
